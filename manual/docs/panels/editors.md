# Editors

The bottom panel displays an editor that auto-switches based on the current selection. Select a MIDI clip and it shows the Piano Roll; select an audio clip and it shows the Waveform Editor, and so on.

| Editor | Shown when you select |
|--------|-----------------------|
| [Piano Roll](piano-roll.md) | A MIDI clip |
| [Drum Grid Editor](drum-grid-editor.md) | A drum clip, or a DrumGrid device is active |
| [Waveform Editor](waveform-editor.md) | An audio clip |
| [Track Chain](track-chain.md) | A track or device |
